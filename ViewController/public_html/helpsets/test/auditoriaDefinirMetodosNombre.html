Cada módulo esta compuesto por eventos, los cuales son ejecutados internamente por el sistema cada vez que el usuario selecciona una opción del mismo. Auditar los eventos le permite registrar con exactitud cada uso que el usuario hace del sistema. Si los nombres de los eventos no son suficientemente claros, lea la descripción de cada uno de ellos para conocer más sobre su funcionalidad. Los metodos marcados como auditables se registran en el sistema y se cargan cada vez que se inicia el servidor de aplicaciones; y son registrados en el Log de este servidor. Igualmente se imprime esta informacion en el log cada vez que se modifica la definicion de eventos auditables
