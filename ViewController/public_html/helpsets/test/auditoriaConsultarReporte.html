Si selecciona la opcion "No", se mostrará un listado con los siguientes campos: Fecha y hora Módulo y Evento Dirección IP Usuario Si selecciona la opcion "Si", se mostrará un listado con los campos antes mencionados adicionando el campo Log; el cual muestra toda la información utilizada en la ejecución del evento. Si el evento retorna información desde la base de datos; en este campo se muestran todos los datos recibidos; en caso que el evento actualice información, o cree nuevos registros en la base de datos, este campo mostrará la información enviada a la base de datos.
