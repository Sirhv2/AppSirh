Direccion IP xxx.xxx.xxx.xxx
Ingrese la direccion IP de la maquina desde la cual se genero el evento. Las direcciones IP tienen el formato XXX.XXX.XXX.XXX donde cada grupo (XXX) debe estar entre 0 y 255.
Ingresando una dirección IP la consulta filtrará todos los eventos generados desde la máquina que tiene asignada esta dirección. Una dirección IP es Única en toda la red del instituto sin embargo debe tener en cuenta que las direcciones IP son asigandas por medio del protocolo DHCP; para asegurarse cual dirección IP tiene asociada una máquina o tuvo asociada una máquina en alguna fecha específica, consulte con el administrador de la red. Las direcciones IP tienen el formato XXX.XXX.XXX.XXX dónde cada grupo (XXX) debe estar entre 0 y 255.
