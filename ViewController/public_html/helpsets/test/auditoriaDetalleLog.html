Detalle del Evento
Informacion que fue afectada con la ejeucion del evento.
Este campo muestra toda la información utilizada en la ejecución del evento. Si el evento retorna información desde la base de datos; en este campo se muestran todos los datos recibidos; en caso que el evento actualice información, o cree nuevos registros en la base de datos, este campo mostrará la información enviada a la base de datos.
